# Project Goals

A tool designed to compare product prices across multiple online stores or marketplaces.
This project helps users find the best deal by automating price retrieval, tracking price changes, and generating insights from multiple data sources.

# Features
Data Extraction
Web scraping from multiple stores
API-based price retrieval
Scheduled price updates

Comparison Engine
Compare prices across vendors
Calculate price differences
Show minimum, average, and maximum price

Price Tracking
Store daily/weekly price history
Generate trend graphs
Detect and notify price drops (optional)

Visual Outputs
Bar charts for store-wise comparison
Line graphs for price trends
Tables summarizing product information
